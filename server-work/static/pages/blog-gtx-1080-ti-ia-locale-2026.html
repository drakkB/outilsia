11 Go VRAM

GTX 1080 Ti : vieux GPU, mais encore un vrai ticket d'entrée IA locale.

La GTX 1080 Ti n'a pas les raffinements récents des RTX, mais ses 11 Go de VRAM restent précieux pour lancer des modèles 7B/8B quantifiés et tester Ollama sérieusement.

Scanner ma GTX 1080 TiComparer RTX 3060

Verdict OutilsIA

OuiMistral 7B, Qwen 8B, Llama 8B, petits modèles code et chat.
À validerHermes 8B, Qwen 14B selon quantization, driver et runtime.
LimitePas de tensor cores modernes, conso/chauffe, VRAM juste sous 12 Go.
UpgradeRTX 3060 12 Go pour moderniser, RTX 3090 24 Go pour changer de palier.

Commandes de départ

ollama run qwen3:8b, ollama run mistral:7b, puis Hermes 8B si la machine reste stable. Le benchmark doit trancher : tokens/s, temps de réponse et qualité en français.

Pourquoi elle reste intéressante

Beaucoup de laptops récents ont 0 à 8 Go VRAM. Une GTX 1080 Ti donne 11 Go dédiés : c'est suffisant pour apprendre l'IA locale, comparer des modèles et éviter de tout faire en CPU.

Ce que le Cockpit doit surveiller

Driver NVIDIA, CUDA visible, température, puissance, RAM système, modèles installés, fréquence mémoire et runtime Ollama Windows/WSL. Si le driver ne remonte pas les métriques fines, OutilsIA doit rester prudent.
