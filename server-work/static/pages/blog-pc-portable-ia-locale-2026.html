Modèles conseillés

Commencez par Qwen 0.6B, Phi mini, Gemma petit ou un 3B quantifié. Un 7B peut être tenté sur certains laptops, mais il doit être présenté comme test de patience, pas comme recommandation quotidienne.

Ollama Windows ou WSL ?

Sur portable Windows, Ollama natif est le plus simple. WSL est utile si vous travaillez déjà dans Ubuntu, mais il ajoute une couche technique. Local Cockpit détecte les deux séparément.

Upgrade réaliste

Si le laptop a 16 Go soudés et aucun GPU, le meilleur upgrade n'est parfois pas un composant : c'est une machine secondaire avec GPU 12 Go VRAM ou un mini PC plus récent. Le rapport OutilsIA doit le dire sans humilier l'utilisateur.
