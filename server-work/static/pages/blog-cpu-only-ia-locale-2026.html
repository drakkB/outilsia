CPU only

IA locale sans GPU dédié : oui pour tester, non pour tout faire.

Un PC CPU only peut lancer de petits modèles locaux, mais il faut rester honnête : l'expérience dépend surtout de la RAM, du processeur et de votre patience.

Modèles conseillés, limites et upgrade utile

Modèles conseillésqwen3:0.6b, Phi mini, Gemma petit, petits GGUF quantifiés.
Limites14B, 32B, image IA lourde, agents longs, gros contextes.
RAM utile16 Go minimum pour découvrir, 32 Go plus confortable.
Upgrade utileRTX 3060 12 Go si vous voulez utiliser Ollama souvent.

Pourquoi commencer par un modèle minuscule

Le premier objectif n'est pas de battre ChatGPT. C'est de vérifier qu'Ollama se lance, qu'un prompt répond, que le benchmark fonctionne et que le rapport OutilsIA reste encourageant. Sur CPU only, un modèle léger donne une preuve locale propre sans faire croire qu'un 32B sera confortable.

Commandes à essayer

ollama run qwen3:0.6b puis un petit modèle 1.5B/3B selon votre RAM. Si la réponse est trop lente, gardez le PC pour découverte et passez au simulateur d'upgrade.

Ce que le Cockpit vérifie

OutilsIA Local Cockpit détecte CPU, RAM, stockage, Ollama, modèles installés et benchmarks. Il signale aussi si la machine devrait rester en mode “portable/vieux PC” ou si un upgrade GPU changerait vraiment l'expérience.
